ArrayList¬∂

ArrayList es una clase del API de Java que es similar a un array y que permite almacenar objetos. A diferencia de una array, el tama√±o de un objeto ArrayList se ajusta autom√°ticamente para acomodar el n√∫mero de elementos almacenados en √©l. Algunas de sus principales caracteristicas son:

  • Un objeto ArrayList se expande autom√°ticamente a medida que se agregan elementos.

  • Adem√°s de a√±adir elementos a un ArrayList, tambi√©n es posible eliminarlos.

  • Un objeto ArrayList se reduce autom√°ticamente a medida que se eliminan elementos de √©l.

Para usar esta clase se requiere la siguiente sentencia de codigo:

import java.util.ArrayList;

Creando un ArrayList¬∂

Sintaxis¬∂

ArrayList<Tipo> nameArrayList = new ArrayList<Tipo>();

Ejemplos

ArrayList<String> nameList = new ArrayList<String>();
ArrayList<Estudiante> al = new ArrayList<Estudiante>();

Usando un ArrayList¬∂

  • Agregar elementos al ArrayList: para agregar elementos se usa el metodo add:

nameArrayList.add(element);

Ejemplo

nameList.add("James");
nameList.add("Catherine");

  • Obtener el tama√±o ArrayList: para obtener tama√±o se usa el metodo size:

nameArrayList.size();

Ejemplo

nameList.size(); // returns 2

  • Obtener un elemento del ArrayList: para obtener un elemento de un ArrayList se usa el metodo get pasando como parametro el indice del elemento a recuperar:

nameArrayList.get(index);

Ejemplo:

nameList.get(1);

import java.util.ArrayList; // Needed for ArrayList class

/**
   This program demonstrates an ArrayList.
*/

public class ArrayListDemo1
{
   public static void main(String[] args)
   {
      // Create an ArrayList to hold some names.
      ArrayList<String> nameList = new ArrayList<String>();
      
      // Add some names to the ArrayList.
      nameList.add("James");
      nameList.add("Catherine");
      nameList.add("Bill");
      
      // Display the size of the ArrayList.
      System.out.println("The ArrayList has " +
                         nameList.size() +
                         " objects stored in it.");

      // Now display the items in nameList.
      for (int index = 0; index < nameList.size(); index++)
         System.out.println(nameList.get(index));
   }
}
