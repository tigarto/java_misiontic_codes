Arreglos¬∂

Definici√≥n¬∂

Un arreglo es una secuencia de datos del mismo tipo llamados elementos los cuales pueden ser accedidos medinte un subindice.

Declaraci√≥n y creaci√≥n de un arreglo¬∂

Declaraci√≥n¬∂

Un array se declara de modo similar a otros tipos de datos; sin embargo, se debe indicar al compilador que es un arreglo y esto se hace con corchetes. La declaraci√≥n se puede hacer de dos formas:

  1. Colocando los corchetes al principio:

tipo [] identificador;

  1. Colocando los corchetes al final:

tipo identificador[];

Ejemplos: A continuaci√≥n se muestran varios ejempls de declraci√≥n:

  1. Declaraci√≥n tipica:

int[] numbers;

O tambien la forma equivalente:

int numbers[];

  1. Declaraci√≥n de varios arreglos en una misma linea:

int[] numbers, codes, scores; // numbers, codes, scores son arreglos tipo int

  1. Notaci√≥n que combina arreglos y datos simples:

int numbers[], codes[], scores; // Solo numbers y codes son arrays tipo int, 
                                // scores es una variable sencilla tipo int.

Creaci√≥n¬∂

Java considera que un arreglo es una referencia a un objeto por lo tanto para que realmente cree un arreglo, es necesario usar el operador new junto altipo de los elementos del arreglo y su n√∫mero.

tipo nombreArreglo [] = new tipo[numeroDeElementos];

En resumen¬∂

Tal y como se mostr√≥ anteriormente, la definici√≥n y creaci√≥n de un arreglo involucra dos pasos tal y como se muestra en el siguiente fragmento de codigo:

int[] numbers;              // 1. Declarar una referencia al array que mantendra los enteros.
numbers = new int[6];       // 2. Crear un nuevo array que mantenga 6 enteros

El proceso anterior (declarar la referencia a un arreglo y crearlo) puede ser realizado en una sola instrucci√≥n tal y como se muestra a continuaci√≥n:

int[] numbers = new int[6]; // Declaraci√≥n y creacion de un array que mantiene 6 enteros

Ejemplo: A continuaci√≥n se muestran varias formas de inicializar un arreglo.

  • Debe ser un numero entero positivo.

  • Puede ser un valor literal, una constante o una variable.

Metodo arraycopy¬∂

Sintaxis:

System.arraycopy (arreglo Origen, inicioOrigen, arreglo Destino, inicioDestino, numElementos)

Donde:

  • arreglo Origen: nombre del arreglo que se va a copiar.

  • inicioOrigen: posici√≥n del arreglo origen donde se inicia la copia.

  • arreglo Destino: nombre del arreglo en el que se hace la copia.

  • inicioDestino: es la posici√≥n del arreglo destino donde empieza la copia.

  • numElementos: n√∫mero de elementos del arreglo origen que se van a copiar.

%%tutor -l java -k

public class CopyExample3 {
    public static void main(String[] args) {
        double [] r, w;
        r = new double[11];
        w = new double[11];
        for (int j = 0; j < r.length; j++) {
            r[j] = (double) 2*j-1;
        }
        // asignaci√≥n del arreglo r a w
        System.arraycopy (r, 0, w, 0, r.length);
        r[3] = 4;
        w[4] = -6;
    }
}

Ejercicio 10.2¬∂

Definir dos arreglos de tipo double, v y w con 15 y 20 elementos respectivamente; en el primero se guardan los valores de la funci√≥n e^(2*x - 1); el segundo inicializa cada elemento al ordinal del elemento; despu√©s se copian los 10 √∫ltimos elementos de v a partir del elemento 11 de w; por ultimo, se escriben los elementos de ambos arreglos.

Soluci√≥n:

public class CopiArreglo
{
    public static void main(String [] a)
    {
        final int N = 15;
        final int M = 20;
        double [] v = new double[N];
        double [] w = new double [M];
        double x = 1.0;

        for (int i = 0; i < N; x+=0.2,i++) {
            v[i] = Math.exp(2 * x - 1);
        }

        for (int i = 0; i < M; i++) {
            w[i] = (double) i;
        }

        // Se imprimen los elementos del vector v
        System.out.println("\n Valores del vector v");
        for (int i = 0; i < N; i++) {
            System.out.print(v[i] + " ");
        }

        // Es realizada la copia de v a w
        System.arraycopy(v, (N-1)-10 +1, w, 10, 10);

        // Se imprimen los elementos del vector w
        System.out.println("\n Valores del vector w");
        for (int i = 0; i < M; i++) {
            System.out.print(w[i] + " ");
        }
    }
}

Bucle for each para recorrido de arreglos y colecciones¬∂

%%tutor -l java -k

public class Bucle {
    public static void main(String[] args) {
        int sumA = 0, sumB = 0;
        int [] a = {1,2,3};
        int [] b = {4,5,6};
        
        // Forma tradicional
        for (int i = 0; i < a.length; i++) {
            System.out.print(a[i] + " ");
            sumA += a[i];
        }
        System.out.print("\nsumA = " + sumA + "\n\n");
        // Forma for each
        for (int num:b) {
            System.out.print(num + " ");
            sumB += num;
        }
        System.out.print("\nsumB = " + sumB + "\n");
    }
}
